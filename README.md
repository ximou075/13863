# Test Results on the iPhone Dataset (Baseline vs. Ours)

We provide results on 5 in-the-wild scenes.

Each folder contains the following 5 videos:

- `input.mp4` — the input provided to the model  
- `render.mp4` — geometric warp of the input from the new camera view (incomplete, contains holes)  
- `mask.mp4` — hole regions (white = to be generated, black = warped reference available)  
- `gen.mp4` — output generated by the diffusion model (guided by render, mask, and reference frames)  
- `viz.mp4` — side-by-side comparison of original vs. generated (looped for easy visualization)
